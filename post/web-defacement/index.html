Apa itu Defacement ?

Di dalam dunia hacking, Defacement umpama tandatangan hacker, meninggalkan jejak ataupun kesan selepas daripada godaman. Siapa sahaja yang tidak mahukan namanya dikenali?, oleh itu para hacker ini meninggalkan beberapa paparan yang menunjukkan informasi atau mesej dari mereka. Itulah yang dinamakan Defacement. Biasanya defacement dibuat pada website.

Defacement bukan sahaja dikhususkan bagi penampilan website bahkan juga boleh dikenakan kepada system os, server dan sebagainya, samaada dalam bentuk text, gambar dan sebagainya.

Script Defacement tidak mempunyai jenis script yang tetap, kerana ada kalanya para hacker ini menggunakan html, javascript, php, asp, png/jpg/jpeg/webb(image). dan sebagainya. Ada juga yang mengenakan file ber-ekstensi *.txt.

Beberapa gambar hasil daripada defacement hacker-hacker yang dikenali ramai 😁. Janganlah anda tersalah dalam menggunakan kuasa dan kemahiran anda. Gunakan ia dalam membantu, bukannya merosakkan.

Sudah fahamkah apa itu defacement?.
